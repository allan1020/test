# Snake Game

A classic Snake game built with React and TypeScript, styled using Tailwind CSS.

## Run Locally

**Prerequisites:** Node.js 18+ and npm

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open the app at the local URL shown in the terminal.

## Build

`npm run build`

## GitHub Pages

This repository is configured to deploy automatically to GitHub Pages from the `main` branch using GitHub Actions.
The site is built to `dist/` and served from `/TEST/`.

## Preview

`npm run preview`
